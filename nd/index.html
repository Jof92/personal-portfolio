OverMij

Front-end ontwikkelaar

Mijn reis in programmeren begon als freelancer en werd gevormd door persoonlijke projecten die me in staat stelden nieuwe technologieën te verkennen en te verbeteren.
Ik studeer Analyse en Ontwikkeling van Systemen, met een verwachte afstudeerdatum in 2024. Gedurende deze reis heb ik mijn leren verbeterd door cursussen te volgen bij erkende technologische scholen zoals Alura, DIO en COD3R.
Mijn carrière als front-end ontwikkelaar omvat professionele en persoonlijke projecten, waarbij ik mijn vaardigheden kan tonen met essentiële technologieën zoals HTML, CSS, JavaScript, React.js en Next.js.
Mijn passie voor games heeft me aangespoord om mijn vaardigheden te ontwikkelen in het maken van games. Een van mijn projecten is geïnspireerd op het universum van Super Mario Bros en is gebouwd met HTML, CSS, JavaScript, en meer.
Om deze projecten en meer te ontdekken, nodig ik je uit om mijn GitHub te bezoeken, waar ik mijn leerervaringen deel.

Mijn Eerste Portfolio

Super Mario Run

Als fan van games en het personage Mario heb ik mijn eigen versie van het spel gemaakt op een eenvoudige manier, met technologieën zoals HTML, CSS en JavaScript. Deze app is een leuke en nostalgische hommage aan dit iconische personage.

Hungry Snake

Dit project is mijn versie van de beroemde mobiele game uit de jaren 2000, Hungry Snake. Het heeft een meer complexe logica en maakt gebruik van technologieën zoals HTML, CSS en JavaScript. Het logo van het spel werd gegenereerd door AI.

Blogproject

Dit project omvat het maken van een blog met behulp van HTML, CSS en JavaScript. JavaScript haalt gegevens op van een JSON API om de lijst met berichten en hun details te laden. Gebruikers kunnen interactie hebben, de details van het bericht bekijken en opmerkingen leuk vinden.
